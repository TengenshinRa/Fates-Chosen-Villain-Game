# Fates-Chosen-Villain-Game

Dies ist ein Xianxia-Kultivierungsspiel.

### Aktueller Stand:
- Basis-Loop (Meditieren & Qi sammeln) funktioniert.
- Durchbruch-Logik implementiert.

### Ziele für Jules:
1. Implementiere ein Kampfsystem gegen "Söhne des Glücks".
2. Erstelle ein Inventar für Pillen und Schätze.
3. Füge eine "Villain-Aura" Mechanik hinzu.
